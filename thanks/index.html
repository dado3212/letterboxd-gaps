Thanks

This website relies on a lot of data, APIs, public libraries, and other repositories. There is no formal affiliation between this website and any of these or their creators, but wanted to say thanks.

Letterboxd

All fonts, colors, and poster images are theirs. Any data mistakes are my own. For rendering the map for the country view I use their version of svgMap.min.js, which in turn is a modified version of StephanWagner's svgMap (see LICENSES).

Also just a huge thanks to them for making such a great website and community, I've discovered and enjoyed so many gems that I wouldn't have without its existence.

TMDb

Letterboxd uses The Movie Database (TMDb), and I also use it for fetching country, language, and female director information. See Letterboxd's page for full information.
This product uses the TMDB API but is not endorsed or certified by TMDB.

Projects/People

  • https://jamesbvaughan.com/movie-director-genders/ - An analysis that James did on his own data (Ruby) that prompted me to make this in PHP
  • https://github.com/GoodbyteCo/Directed-By-Women - A broken website that tried to do just the "directed by women" section
  • https://colorboxd.com/ - A (currently) broken website for color sorting lists that sparked my curiosity in color sorting (though most of that code hasn't yet landed in this repo)
  • Kirsten Dunst's pitch for actresses to work with female directors - "It's up to us as actresses to give the opportunity to first time directors. Most of those people it was their first time." Being more thoughtful about watching and highlighting movies by female directors is a tiny contribution, but one that I wanted to be more conscious about.
